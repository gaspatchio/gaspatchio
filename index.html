Gaspatchio vs Lifelib — IntegratedLife VA Model --

Both engines produce identical results — 0.0000% reconciliation across 9 million data points. This comparison measures execution speed only. lifelib is designed for formula transparency and auditability via modelx's cell-based execution model. Gaspatchio is designed as an LLM-first actuarial framework — its primary goal is enabling AI agents to build and run actuarial models, with high performance as a natural consequence of its Rust/Polars architecture. These are complementary tools optimizing for different goals. Learn more about lifelib at lifelib.io

Aggregation Surface

Three ways to aggregate a scenario projection on the Level 4 model, all producing identical results: baseline materialises every projection then aggregates, run_aggregated folds results on the fly, and spill streams to disk. The trade-off is peak memory versus throughput — run_aggregated holds throughput while cutting peak memory several-fold.
